**MFS CoopCareer** is a cooperative career mode for Microsoft Flight Simulator 2024.

This repository is the public distribution point for official Windows builds and update files. The project source code is maintained in a separate private repository and is not published here.

> [!IMPORTANT]
> For a normal installation, download the complete package:
>
> **`MFS-CoopCareer-VERSION-win-x64.zip`**
>
> from the latest stable release.

### [Download the latest stable release](https://github.com/KickerMix/MFS-CoopCareer-Releases/releases/latest)

Previous versions are available on the [Releases](https://github.com/KickerMix/MFS-CoopCareer-Releases/releases) page.

## Installation

1. Open the [latest stable release](https://github.com/KickerMix/MFS-CoopCareer-Releases/releases/latest).
2. Download:

   ```text
   MFS-CoopCareer-VERSION-win-x64.zip
   ```

3. Completely close Microsoft Flight Simulator 2024.
4. Extract the archive into a separate folder.

   Do not run the application directly from the ZIP archive.

5. Open the extracted folder and follow the instructions in:

   ```text
   README_RU.txt
   ```

6. Start the required components as described in the included instructions.

> [!TIP]
> Installing the application into protected system directories such as `C:\Program Files` is not recommended.
>
> Use a separate folder instead, for example:
>
> ```text
> C:\Games\MFS-CoopCareer
> ```

## Updates

Starting with version **1.9.0**, stable updates are checked and installed directly through the Companion application.

To update MFS CoopCareer:

1. Start Companion.
2. Wait for the update check to complete.
3. Confirm the installation when a new version is available.
4. Do not close the application until the update process has finished.

The separate `Companion`, `Server`, and `Community` archives published with a release are used by the built-in updater.

They do not need to be downloaded or installed separately for a normal clean installation.

## Release Files

The exact contents may vary between releases, but a typical release includes:

| File | Purpose |
|---|---|
| `MFS-CoopCareer-VERSION-win-x64.zip` | Complete portable package for a clean installation |
| `Companion` archive | Companion component used by the built-in updater |
| `Server` archive | Server component used by the built-in updater |
| `Community` archive | Community package used by the built-in updater |
| `update-manifest.json` | Update manifest used by Companion |
